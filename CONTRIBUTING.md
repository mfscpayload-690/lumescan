# Contributing to LumeScan

Thank you for your interest in contributing to LumeScan!

## How to Contribute

1. Fork the repository.
2. Create a new branch for your feature or bugfix.
3. Make your changes and ensure they follow the project's style.
4. Submit a pull request with a detailed description of your changes.

## Development Setup

- **Web**: `cd web && npm install && npm run dev`
- **Server**: `cd server && python -m venv venv && source venv/bin/activate && pip install -r requirements.txt`
